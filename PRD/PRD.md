# FA Campaign Manager PRD

## 1. 프로젝트 개요
TRPG(Tabletop Role-Playing Game) 캠페인을 관리하는 백엔드 애플리케이션으로, 캠페인, 캐릭터, 세션 및 유저 관리를 통합적으로 제공합니다.

## 2. 핵심 목표
- 캠페인 및 연관 데이터(세션, 캐릭터, 멤버)의 통합 관리
- 확장 가능하고 안전한 RESTful API 제공
- 코드 품질 및 유지보수성을 위한 클린 아키텍처 및 계층화 준수

## 3. 주요 기능
- 사용자 인증 및 관리
- 캠페인 생성/수정/삭제 및 참여
- 캠페인 내 세션 및 캐릭터 관리
