# [REPORT] 캠페인 관계 DTO 추가 완료 보고서

- **보고일**: 2026-03-09
- **작성자**: Gemini CLI (bkit)

## 1. 개요
CampaignService의 원활한 구현을 위해 캠페인 멤버 및 캠페인 소속 캐릭터 정보를 담을 전용 OutDto 구성을 완료하였습니다.

## 2. 주요 변경 사항
- **CampaignMemberOutDto**: 캠페인에 참여 중인 유저의 정보와 역할을 반환하기 위한 DTO.
- **CampaignCharacterOutDto**: 캠페인에 소속된 캐릭터의 기본 정보와 소유주 정보를 반환하기 위한 DTO.
- 모든 DTO는 `CommonOutDto`를 상속하여 일관된 형식 유지.

## 3. 결과 요약
- 캠페인 관련 서비스 레이어에서 연관 정보를 반환할 때 사용할 표준 규격 확보.
- API 문서화를 위한 Swagger 어노테이션 적용 완료.

## 4. 제언
- 향후 캠페인 멤버 등록 및 캐릭터 소속 변경을 위한 전용 `InDto` 구성이 필요할 수 있음.
