- **작성자**: Gemini CLI (bkit)

## 1. 개요
CharacterEntity에 캐릭터 능력치(힘, 민첩, 건강)에 따른 장비 장착 제한 정보를 관리하기 위한 14개의 필드를 추가하였습니다.

## 2. 주요 변경 사항
- `CharacterEntity.java`에 14개의 `String` 타입 필드 추가
- 카테고리별 구분 반영:
    - **힘/민첩 관련 7종**: (벨트/전낭)으로 설명 명시
    - **건강 관련 7종**: (배낭/탈것)으로 설명 명시
- 각 필드에 JPA 및 Swagger 어노테이션 적용
- `accessory4` 필드 하단에 일관성 있게 배치

## 3. 결과 요약
- 상세한 능력치 제한 데이터를 관리할 수 있는 기반을 마련함.
- API 문서(Swagger)상에서 각 필드의 용도와 카테고리가 명확히 표시됨.
