# 중요 규칙.

- **CRITICAL**: 사용자의 요구를 수행하기 전, 작업중인 엔드포인트 폴더의 모든 파일, common 의 모든 파일을 조회하고 시작합니다.
- **CRITICAL**: 한국어로 답변합니다.
- **CRITICAL**: 컨트롤러에 엔드포인트 메소드 어노테이션 순서는 다음과 같습니다.
  1. @IsUser @IsAdmin 등등
  2. @GetMapping 등등
  3. @Operation. summary, description 만 적용.
  4. @ApiResponses 로 응답 예시 별도로 작성.
- **CRITICAL**: 모든 리소스 반환에는 테이블 관계에 따른 연관 데이터들도 같이 반환합니다.
  - 사용자 조회 기준으로 이야기하면 사용자 조회를 하면 캐릭터 리스트, 참여 캠페인 리스트, 참여 세션 리스트 가 같이 내려와야 합니다.
  - 캠페인 조호 기준으로 이야기 하면 해당 캠페인의 플레이어 리스트, 캐릭터 리스트, 존재 세션 리스트가 같이 내려와야 합니다.
  - 즉, outDto와 Entity 의 관계 컬럼/필드가 동일하게 존재해야 함.
- **CRITICAL**: 클래스를 가져올 때 풀패키지를 적지 않습니다.
